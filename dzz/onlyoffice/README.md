### DzzOffice小胡版安装和升级onlyoffice插件说明：
**重点：小胡版插件基于dzzoffice小胡版编辑，其他版本不确定兼容性。**

1. 下载解压后，将onlyoffice文件夹放在网站dzz\下(如果该目录下已有相同文件夹请先删除在放);

2. 升级的用户在系统工具中清理缓存即可;

3. 安装的用户，在应用市场右下角选择导入，选择导入文件，导入文件是文件夹下的dzz_app_onlyoffice.xml，然后选择提交。
